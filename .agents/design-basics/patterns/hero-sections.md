# Hero Section Patterns

## Pattern 1: SaaS/Tech Hero (Recommended for Your Startup)

```
┌────────────────────────────────────────────┐
│ NAV BAR                                    │
├────────────────────────────────────────────┤
│                                            │
│  [Badge: "New - AI Agent Beta"]           │
│                                            │
│  Headline: Your All-in-One Business OS     │
│  Subhead: CRM, CMS, AI agents — ek platform│
│           mein sab kuch                     │
│                                            │
│  [Get Started Free →]  [Watch Demo]        │
│                                            │
│  Trusted by 500+ teams                     │
│  [Logo1] [Logo2] [Logo3] [Logo4]           │
│                                            │
│          [Dashboard Mockup Image]          │
└────────────────────────────────────────────┘
```

**Specs:**
- Full-bleed background (dark gradient or light)
- Headline: 48-64px, bold, max 10 words
- Subhead: 20-24px, regular, max 20 words
- CTA: Primary (filled) + Secondary (outlined)
- Social proof below CTA (logos + count)
- Background: subtle gradient, mesh gradient, or animated

## Pattern 2: Bold Feature Hero

```
┌────────────────────────────────────────────┐
│ NAV BAR                                    │
├────────────────────────────────────────────┤
│  ┌──────────┐  ┌──────────────────────────┐│
│  │           │  │                          ││
│  │ Headline  │  │                          ││
│  │           │  │   Hero Visual            ││
│  │ Subhead   │  │   (Dashboard UI /        ││
│  │           │  │    Product Screenshot)   ││
│  │ [CTA]     │  │                          ││
│  │           │  │                          ││
│  └──────────┘  └──────────────────────────┘│
│                                            │
│  Stats Bar: 10K+ Users ★ 99.9% Uptime ★ 4.8★│
└────────────────────────────────────────────┘
```

**Specs:**
- Split-screen: 50% text + 50% visual
- Left: headline + subhead + CTA + social proof
- Right: product screenshot / dashboard mockup / 3D visual
- Responsive: stacks vertically on mobile

## Pattern 3: Centered Minimal Hero

```
┌────────────────────────────────────────────┐
│                                            │
│              Headline (centered)           │
│                                            │
│           Subheadline (centered)           │
│                                            │
│             [Primary CTA Button]           │
│                                            │
│         Social Proof / Stats Below         │
│                                            │
└────────────────────────────────────────────┘
```

**Specs:**
- Everything centered
- Max width: 720px content area
- Full-bleed background
- Best for: product launches, simple value props
- Responsive: stays centered on all sizes

## Hero Design Rules

1. **Single message** — ek headline mein puri value prop
2. **Single primary CTA** — doosra optional hai (secondary style)
3. **Visual reinforces message** — don't use random hero image
4. **Mobile CTA = sticky bottom** — always visible
5. **Hero height** — 80-100vh (full viewport on desktop, auto on mobile)
6. **No nav links in hero area** — focus on CTA

## Mobile Hero Adjustments

| Element | Desktop | Mobile |
|---------|---------|--------|
| Headline | 48-64px | 28-36px |
| Subhead | 20-24px | 16-18px |
| Layout | Side-by-side | Stacked (visual below text) |
| CTA | Inline | Full width, sticky bottom |
| Visual | Right side | Below text, full width |
| Padding | 120px top | 48px top |

## Common Mistakes

- ❌ Headline too long (>10 words)
- ❌ Multiple CTAs competing for attention
- ❌ Generic stock photo as hero visual
- ❌ No social proof — visitors don't trust yet
- ❌ Mobile not checked — hero breaks on 360px
