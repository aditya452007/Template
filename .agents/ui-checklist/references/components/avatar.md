# Avatar Checklist

- [ ] **Visualiser** — Show what the avatar looks like before posting.
- [ ] **Link to upload or select avatar** — Clickable with editing indicator (pencil icon).
- [ ] **Placeholder image** — Default icon or user initials.
- [ ] **Acceptable file types** — JPEG, PNG, SVG.
- [ ] **Status change updating avatar** — Banner or toast confirmation.
- [ ] **Alternatives** — Illustrative option if user doesn't want to upload.
- [ ] **Crop and resize** — Allow cropping and resizing before saving.
