# Card Checklist

- [ ] **Style** — Background, border, shadow.
- [ ] **Consistency** — One base style for all cards.
- [ ] **Spacing** — Framework for padding levels (divisible by 4 or 5).
- [ ] **Responsiveness** — Top-down approach on mobile to avoid overfilled rows.
- [ ] **Content hierarchy** — Primary actions clear. Links at end of card content.
