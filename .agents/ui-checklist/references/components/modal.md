# Modal Checklist

- [ ] **Title** — Clear, simple text explaining the action.
- [ ] **Actionable item** — Button/link to continue or close. Clear button copy or descriptive label.
- [ ] **Close action** — Way to exit the modal.
- [ ] **Responsiveness** — Full-screen on mobile.
- [ ] **Background change behind modal** — Darken, blur, or lighten to bring focus.
- [ ] **Description** — Extra information for decision-making.
