# Slider Checklist

- [ ] **Style** — Visual design of range and handlers.
- [ ] **Intervals** — Continuous (any value) or stepped (fixed values).
- [ ] **Handles** — Elements that users drag. Should clearly break out from the container.
- [ ] **Labelling** — Text conveying the spectrum and interval values.
- [ ] **Interaction states** — Handle and slider changes during value adjustment.
