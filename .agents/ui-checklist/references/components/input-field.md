# Input Field Checklist

- [ ] **Input field** — Interactive text field, readable on all devices.
- [ ] **Label** — States what information to provide. Avoid all caps.
- [ ] **Placeholder text** — Example of what to enter. Colour must be lighter than default text.
- [ ] **Data format** — Restrict to relevant values (numeric for phone). Hint at format.
- [ ] **Illustration or icon** — Visual cue to break up long forms.
- [ ] **Hint** — Elaborate on the label if users may struggle.
