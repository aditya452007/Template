# Verifying Account Flow Checklist

- [ ] **Establish a trigger point** — Clear indication when and why verification is required.
- [ ] **Method selection (optional)** — Multiple verification methods (email, SMS, authenticator).
- [ ] **Confirm delivery and contact information used** — Display destination (email/phone) for user confirmation.
- [ ] **Ability to input verification code** — Intuitive input (one field per digit or standard field).
- [ ] **Incorrect value (and resend option)** — Specific error messages for expired/invalid/max-attempts. Clear next steps.
- [ ] **Verification success state** — Clear confirmation and transition to next step.
