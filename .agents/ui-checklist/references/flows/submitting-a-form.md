# Submitting a Form Flow Checklist

- [ ] **Show button to submit** — Below form fields. Copy matches form purpose ("Subscribe", "Create Account").
- [ ] **Show loading state after submission** — Loading indicator. Consider hover state before press.
- [ ] **Show success message when it submits** — Clear success confirmation.
- [ ] **If it doesn't, show an error message** — Clear error communication.
- [ ] **Error may occur because of wrong information** — Detail which field failed and why.
