# Login Page Checklist

- [ ] **Logo** — Complete logo or symbol mark.
- [ ] **Title** — Tell the user this is the login page.
- [ ] **Account identification** — Username, email, or phone number. Consider multiple identifiers.
- [ ] **Password** — Secure input with visibility toggle.
- [ ] **Link to reset password** — Positioned near the password field.
- [ ] **Link to sign up** — For users without an account.
- [ ] **Login via third party** — Google, Apple, Facebook, Twitter, etc.
- [ ] **Testimonial** — Brand reinforcement.
- [ ] **Blog post** — Relevant content showcase.
- [ ] **New feature** — Build excitement before login.
