# Search Checklist

- [ ] **Search box** — At top of page, styled consistently. Include submit button and helpful placeholder text.
- [ ] **Visualisation of results** — Show title, description, thumbnail. Display result count. Include pagination for 50+ results. Handle zero results with guidance.
- [ ] **Search progress** — Loading indicator for non-instant searches.
- [ ] **Suggested search items** — Trending, popular, recent searches, autocorrection.
- [ ] **Categories and/or filters** — Constraints to narrow down search area.
- [ ] **Ordering of search results** — Sort by price, relevance, newest, etc.
- [ ] **Highlighting keywords** — Highlight search terms in results.
