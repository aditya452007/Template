# Typography — Fonts Ki Duniya

> Source: Figma "Best Fonts for Websites 2026" + "Typography" articles

## Core Principles

- **Max 2 font families per project** — 1 heading + 1 body
- **Sans-serif for UI** (cleaner, more readable on screens)
- **Serif for headings/print/luxury** (authority, tradition)
- **16px minimum for body text**
- **40-60 characters per line** for optimal readability
- **Line height: 1.5 body, 1.2 headings**

## Font Hierarchy

| Level | Weight | Size | Usage |
|-------|--------|------|-------|
| Display | Black (900) | 64-96px | Hero headlines, big impact |
| H1 | Bold (700) | 48-64px | Page titles |
| H2 | Bold (700) | 32-48px | Section headers |
| H3 | Semi-Bold (600) | 24-32px | Card titles, sub-sections |
| H4 | Medium (500) | 20-24px | Minor headings |
| Body L | Regular (400) | 18px | Large body |
| Body | Regular (400) | 16px | Standard body |
| Small | Regular (400) | 14px | Secondary text |
| Caption | Medium (500) | 12px | Labels, badges |

## Best Web Fonts 2026

**Sans-serif (UI/General):**
- **Inter** — GitHub, Mozilla use karte hain. Clean, neutral, versatile
- **Roboto** — Android ka default. Friendly, geometric
- **Open Sans** — Highly readable. Good for text-heavy pages
- **Lato** — Warm, professional. Good balance
- **Montserrat** — Urban, bold. Good for headlines
- **Manrope** — Modern, variable. Good for tech startups
- **Plus Jakarta Sans** — Trendy, geometric. Good for AI/modern brands

**Serif (Luxury/Headings):**
- **Merriweather** — Serious, authoritative
- **Playfair Display** — Elegant, dramatic. Good for luxury headings
- **Source Serif 4** — Clean serif. Good for long-form
- **DM Serif Display** — Modern serif. Good for paired with sans body

**Monospace (Code/Data):**
- **JetBrains Mono** — Developer favorite
- **Fira Code** — With ligatures. Good for code snippets

## 5 Common Typography Mistakes

1. ❌ Using too many fonts (3+)
2. ❌ Body text too small (< 16px)
3. ❌ Poor line height (too tight or too loose)
4. ❌ Forgetting responsive font sizes
5. ❌ Using all caps for long text
