# Design Basics Skill — Figma Resource Library se Liya Gya Gyaan

## Ye Skill Kya Karti Hai?

Ye skill **Figma Resource Library ke 122+ design articles** ka saransh hai. Jab bhi tum koi website ya UI design generate karte ho, ye skill ensure karegi ki design **professional, accessible, aur conversion-optimized** ho. AI ko "ugly design" se bachane ke liye concrete rules aur guardrails diye gaye hain.

## Kab Use Karein?

- Koi bhi naya website design generate karne se pehle
- Color palette, typography, ya layout decide karte waqt
- Pricing page, landing page, ya portfolio design karte waqt
- Design trends 2026 ke baare mein puchne par
- Koi bhi UI component (buttons, forms, navigation) design karte waqt
- AI-assisted design ya vibe coding karte waqt

## Quick Start — 5 Second Rule

Design generate karne se pehle ye 5 cheezein check karo:

1. **Pehle `guardrails.md` padho** — isme hai ki kaise AI ugly design se bachega
2. **Brand vibe decide karo** — playful? luxurious? professional? bold? minimal? (tumhara startup mix hai — har section ke hisaab se sahi vibe choose karo)
3. **Mobile-first socho** — 51.4% spending mobile se hoti hai
4. **Contrast check karo** — 79.1% homepages fail karte hain contrast mein
5. **Single CTA per page** — ek page, ek goal, ek action

## General Design Principles (Har Module Mein Common)

- **Users form opinion in 50 milliseconds** — first impression matters
- **94.8% of top 1M homepages fail WCAG** — accessibility ko ignore mat karo
- **1-sec load = 2.5x higher conversion than 5-sec** — performance matters
- **52% users stop after one bad experience** — design directly business impact karta hai
- **71% consumers expect personalization** — generic design chhod do
