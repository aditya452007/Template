# UX Copy Checklist

- [ ] **Clear and concise** — Say exactly what will happen in as few words as possible.
- [ ] **Purposeful** — Every word has intent. User understands action before interaction.
- [ ] **Tone** — Consistent with brand. Some brands benefit from humour, others don't.
- [ ] **Conventions** — Maintain one term per action (don't swap "Next" and "Continue").
- [ ] **Avoid dark patterns** — No sarcasm, antagonism, or guilt tripping to influence decisions.
