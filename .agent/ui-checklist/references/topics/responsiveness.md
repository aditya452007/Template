# Responsiveness Checklist

- [ ] **Fluid layout** — Content accessible on any device. Grids and columns for proportional content.
- [ ] **Breakpoints** — Identify key points: mobile, tablet, desktop, ultra wide.
- [ ] **Typography** — Scale text sizes across breakpoints (H1 differs on desktop vs mobile).
- [ ] **Selection area for actions** — Larger touch targets on mobile, finer accuracy on desktop.
- [ ] **Hierarchy** — Adapt content and visual structure per screen size.
