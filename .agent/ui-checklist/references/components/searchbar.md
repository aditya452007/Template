# Searchbar Checklist

- [ ] **Input field** — Clear container for typing.
- [ ] **Label or placeholder text** — Identify purpose. Suggest search examples.
- [ ] **Quick links, autocomplete and suggestions** — Offer based on typed input or past searches.
- [ ] **Submit search button** — Visible action with loading indicator for connection issues.
- [ ] **Previous searches** — Speed up repeat queries.
- [ ] **Search scope** — Whole platform or specific area.
