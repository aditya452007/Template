# Drawer Checklist

- [ ] **Placement** — Which edge the drawer slides from (left for navigation, right for details/settings).
- [ ] **Dimensions** — Full height, width sufficient for content without taking over the entire page.
- [ ] **Overlay** — Semi-transparent background to bring focus to the drawer.
- [ ] **Header** — Title describing contents, optional close action and description.
- [ ] **Content area** — Scrollable main section.
- [ ] **Open and close trigger** — Button to open; overlay click, close button, or Esc key to close.
- [ ] **Footer** — Fixed bottom area for primary/secondary actions (Save & Cancel).
