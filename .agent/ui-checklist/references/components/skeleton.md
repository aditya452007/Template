# Skeleton Checklist

- [ ] **Match content structure** — Mirror size and layout of actual content.
- [ ] **Animation** — Subtle shimmer, pulse, or wave motion.
- [ ] **Style** — Flat shapes, rounded or sharp corners.
- [ ] **Color scheme** — Neutral gray or soft colour, just enough contrast with background.
- [ ] **Transition to real content** — Smooth fade out before content loads.
