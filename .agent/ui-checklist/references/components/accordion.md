# Accordion Checklist

- [ ] **Header** — Clickable trigger with title that clearly conveys content inside.
- [ ] **Expand/collapse icon** — Plus/minus or caret/chevron that rotates.
- [ ] **Content area** — Shows/hides on toggle; can contain text, image, video, etc.
- [ ] **States** — Default (collapsed), expanded, hover, focused, disabled.
- [ ] **Expanding logic** — Multiple sections open at once vs one at a time.
