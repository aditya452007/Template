# Navigation Checklist

- [ ] **Logo** — Reinforce brand on every page.
- [ ] **Structure** — Most important links at high level. Two levels max hierarchy. Distinct visual design per level.
- [ ] **Consistent location** — Always at the same position on every page.
- [ ] **Visual contrast to rest of page** — Clearly distinguishable. Consider shadow/background change on scroll.
- [ ] **Call to action/s** — Primary actions structured by importance.
- [ ] **Mobile responsiveness** — Dedicated mobile menu design.
- [ ] **Icons next to link** — Visual cue to break up text.
- [ ] **Breadcrumbs** — Show current level and hierarchy above.
