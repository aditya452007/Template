# Radio Checklist

- [ ] **Label** — Text associated with the radio input.
- [ ] **Grouping** — Must have more than one option. Vertical alignment preferred for longer labels.
- [ ] **Default selection** — Start with a likely option selected based on research.
- [ ] **Style** — Unique from other inputs.
- [ ] **Clickability** — Clickable text label and container.
- [ ] **States** — Default, active, disabled.
