# Icon Checklist

- [ ] **Responsiveness** — Detail level appropriate for all display sizes.
- [ ] **Visual style consistency** — Same stroke weight, corner radius, optical sizing.
- [ ] **Color** — Black/white, flat colors, or gradients. Brand-consistent.
- [ ] **Naming** — Name by what it literally is, not by action (e.g. "pencil" not "edit").
