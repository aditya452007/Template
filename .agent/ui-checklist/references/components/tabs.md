# Tabs Checklist

- [ ] **Labels** — Logical enough to predict tab content. Keep concise, ideally 1 word.
- [ ] **Content area** — Below horizontal tabs or to the right of vertical tabs.
- [ ] **Style** — Active vs inactive visual differentiation.
- [ ] **Item order** — Ordered by popularity or familiarity.
- [ ] **States** — Default, active, hover.
