# Table Checklist

- [ ] **Table header** — Column values providing structure. Fix header on scroll.
- [ ] **Row style** — Borders and contrasting background colours for differentiation.
- [ ] **Spacing** — Consistent padding for each row and header.
- [ ] **Search** — Find specific keywords or rows across all data points.
- [ ] **Actions** — View, edit, delete per row.
- [ ] **Filter and sort** — Customize visible data and ordering.
- [ ] **Responsiveness** — Collapse to accordion or horizontal scroll on small devices.
- [ ] **Pagination** — Breakpoints for digesting large data sets.
