# Banner Checklist

- [ ] **Style** — Fill, text colour, border radius. Must stand out from default UI.
- [ ] **Content** — Title is essential; optional description.
- [ ] **Types** — Information, success, warning, error, neutral.
- [ ] **Call to action button** — Optional, only if user action is needed.
- [ ] **Placement** — Page position based on context and importance.
- [ ] **Dismissable** — Positive/neutral banners can be dismissed; warning/error should not.
