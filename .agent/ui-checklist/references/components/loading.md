# Loading Checklist

- [ ] **Visual indicator** — Clear representation that content is loading.
- [ ] **Text** — Explain the loading state. Be specific ("syncing your emails", not just "loading").
- [ ] **Time** — Determine delay threshold for showing loading component.
- [ ] **Accessibility** — Visible on any background. Container with appropriate contrast.
- [ ] **Visuals** — Entertain user with illustration during load.
