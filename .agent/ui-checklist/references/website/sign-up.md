# Sign Up Checklist

- [ ] **Logo** — Complete logo or symbol mark.
- [ ] **Title** — Tell the user this is where they create an account.
- [ ] **Description** — Basic features and expectations of account creation.
- [ ] **Account identification** — Email is most common, also enables communication.
- [ ] **Setting a password** — Include password strength indicator, requirements, and visibility toggle.
- [ ] **Link to login** — For users who already have an account.
- [ ] **Sign up via third party** — Facebook, Google, etc.
- [ ] **Current active customer count** — Social proof with FOMO.
- [ ] **Testimonial or social proof** — Brand reinforcement.
- [ ] **Blog post** — Show relevant content to convert unsure leads.
