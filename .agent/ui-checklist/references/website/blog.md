# Blog Checklist

- [ ] **Hero** — Clear indicator it's a blog.
- [ ] **Navigation** — Logo, links, CTA, search function. Sub-topic navigation.
- [ ] **Layout** — Consider above-the-fold on mobile, tablet, desktop. Newest-first ordering. Grid layout for density.
- [ ] **Blog post preview** — Clear heading hierarchy, branded thumbnail, concise description.
- [ ] **Description** — Summary of blog in the hero.
- [ ] **Tags and/or categories** — Filterable topics.
- [ ] **Search function** — For large blog archives.
- [ ] **Opt-in for subscription** — Notify users of new posts.
