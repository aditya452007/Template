# Pricing Checklist

- [ ] **Pricing options** — Subscription plans or one-off purchase.
- [ ] **Pricing features** — What the user gets with each option.
- [ ] **A free pathway to sign up** — Trial period with clear length and billing terms.
- [ ] **Refund or return policy** — Clear terms.
- [ ] **Highlighted price as a recommendation** — Most popular or best-value option highlighted.
- [ ] **Security logo for payment processing** — Credible vendor logo.
- [ ] **Frequency of payments (monthly vs yearly)** — Annual discount option.
