# Billing Checklist

- [ ] **Payment method/s used** — Display payment method details with ability to update or remove. Subscription services should prevent all methods being removed.
- [ ] **History** — List of past charges with ability to download invoices (PDF format).
- [ ] **Plan details** — Current plan info includes price, features, and next billing date. Clear action to upgrade or change plan.
- [ ] **Promo or discount code** — Field to apply a code to activate a discount or deal.
- [ ] **Additional details** — Info that influences billing decisions (seats, roles, usage, metrics).
- [ ] **Next payment** — Upcoming payment details (amount, date, method).
- [ ] **Cancel subscription** — Clear action to cancel and stop payments.
