# Blog Post Checklist

- [ ] **Title** — Educational, eye-catching copy.
- [ ] **Sub heading** — Elaborates on the title.
- [ ] **Thumbnail** — Branded visual aid.
- [ ] **Author** — Writer attribution, linked to their profile.
- [ ] **Content** — Blog post body, responsive on all devices.
- [ ] **Social sharing tools** — Fixed-position share buttons.
- [ ] **Lead conversion content** — Product info tied to the post.
- [ ] **Related articles** — Suggested relevant posts.
- [ ] **Opt-in for subscription** — Signup form for new post notifications.
- [ ] **Date posted** — Publication date.
