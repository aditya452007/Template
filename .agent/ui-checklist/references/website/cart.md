# Cart Checklist

- [ ] **Item name and image** — Exactly as on the item page.
- [ ] **Change or remove item** — Edit items inside the cart.
- [ ] **Item variations** — Size, colour, style, type with alternative shipping methods and costs.
- [ ] **Apply promo, discount or coupon** — Obvious text field above checkout button, with success/decline status.
- [ ] **Accepted payment methods** — Secure payment provider logos to instill trust.
- [ ] **Link to support** — For issues or questions about items.
- [ ] **Link to checkout** — Standout action to proceed to purchase.
- [ ] **Suggested items to add** — Upsell relevant items.
