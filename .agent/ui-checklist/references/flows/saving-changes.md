# Saving Changes Flow Checklist

- [ ] **Show action that enables change** — Button triggers editable state if not auto-editable.
- [ ] **Disable save action until changes are made** — Disabled initially, activates on change. Fixed position recommended.
- [ ] **State changes to active once a change is made** — Brings attention to the save action.
- [ ] **Action changes to loading state when pressed** — Loading spinner in the action.
- [ ] **Notify changes have been saved** — Clear confirmation that data is locked in.
