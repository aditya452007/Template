# Canceling Subscription Flow Checklist

- [ ] **Show a link in account details** — In account page or billing settings. Clearly visible.
- [ ] **Confirm intent to cancel** — Ask for confirmation. Optional last-chance value reminder (not pressuring).
- [ ] **Request a reason for canceling (optional)** — Useful feedback for identifying churn factors.
- [ ] **Confirm subscription has been canceled** — Tell user when their subscription is active until.
