# Uploading Media Flow Checklist

- [ ] **Empty state** — Visual placeholder with upload icon and text. Offer click-to-upload alternative.
- [ ] **Drag and drop interaction** — Clear visual state change when file is detected over the area.
- [ ] **Progress indicator** — Real-time progress. Show file names and overall progress for multiple files.
- [ ] **File restrictions & constraints** — Size and format limits shown upfront. Error message for invalid files.
- [ ] **Outcome status** — Green checkmark for success, red warning for failure with explanation.
- [ ] **Upload actions** — Retry, cancel, delete, rename. Default vs hover visibility.
- [ ] **Showing multiple uploaded files** — Clean list or grid with thumbnails. Relevant file info shown.
