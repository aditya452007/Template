# Entering Promo Code Flow Checklist

- [ ] **Show promo code input field** — In cart or checkout, visible near the continue button.
- [ ] **Promo code applied successfully** — Success state colours (green). Hide input if only one code allowed.
- [ ] **Reflect promo code impact** — Show discount as total amount or per-item. Repeat near checkout total.
- [ ] **Promo code error** — Not applicable or incorrect code handling.
- [ ] **Remove promo code** — Allow user to revert to original price or use a different code.
