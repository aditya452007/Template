# Deleting Account Flow Checklist

- [ ] **Show a link to delete account** — In profile/settings and support area. Make it accessible.
- [ ] **Politely ask for feedback** — Request reason for leaving to improve experience. Not pressured.
- [ ] **Explain what it means before confirming** — Data deletion, restoration possibility, deletion timeline.
- [ ] **Confirm account has been deleted** — Acknowledge departure gracefully. No passive aggressiveness.
