# Resetting Password Flow Checklist

- [ ] **Place reset link close to password field** — Styled as a clickable link.
- [ ] **Ask for account details to verify** — Email address (pre-fill from login page if available).
- [ ] **Show information has been sent** — Explain how to continue (email with link, or code to phone).
- [ ] **The message sent explains next steps** — Link to reset page or verification code entry.
- [ ] **Reset the password!** — Clear text field for new password. Guidelines for strength requirements.
- [ ] **Password successfully reset** — Indicate success and direct user to log in.
