# Filtering Items Flow Checklist

- [ ] **Show action near item collection** — Above or beside the collection. Recognizable icon and/or label.
- [ ] **Show available filter options** — Same page (immediate feedback) or separate page (large filter sets). Priority order.
- [ ] **Consider different filter types** — Sliders, checkboxes, dropdowns. Pick the easiest for each property.
- [ ] **Show active filters clearly when applied** — Display applied filters on collection page.
- [ ] **Provide easy filter removal** — Clear individual filters or all filters at once.
- [ ] **Show result count** — Help users gauge if result set is too narrow or broad.
- [ ] **Empty state** — Explain no results and suggest adjusting/clearing filters.
